The Food Waste Management System is a web-based application designed to minimize food waste by connecting food donors (restaurants, hotels, supermarkets) with receivers (orphanages, shelters, NGOs) through volunteers. The system ensures efficient coordination, transparency, and secure handling of food donation activities using a database-driven architecture.

Features

User registration and authentication with role-based access control (admin, donor, receiver, volunteer).

Donation management including food details, quantity, expiry date, location, and pickup schedule.

Request management allowing receivers to browse available donations, place requests, and track their status.

Automated workflows using database triggers and scheduled events to handle food expiry and status updates.

Admin dashboard for monitoring system activity, managing users, and viewing overall statistics.

Tech Stack

Backend: PHP

Database: MySQL (tables, views, triggers, stored procedures, events)

Frontend: HTML, CSS, Bootstrap, JavaScript# Database-Project
